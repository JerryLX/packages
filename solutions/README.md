## <a name="1">Introduction</a>
The project is a micro-ecommerce project base on spring cloud. Up to now, it includes 5 services : api-gateway, eureka, order, cart, search.

## <a name="2">Build</a>
$ mvn clean package

## <a name="3">Installation</a>
$ mvn clean install

## <a name="4">Start</a>
$ mvn spring-boot:run
